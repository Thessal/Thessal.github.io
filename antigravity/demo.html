Scale-Free Network Stability & Phase Transitions

BA Model Equation

The Barabási-Albert model uses preferential attachment. The probability Π(ki) that a new node connects to existing node i is:

Π(ki) = ki / Σj(kj)

Where ki is the degree of node i. m represents the number of links every new node forms. Higher m creates denser meshes.

Why is the graph static? (m=1)

When m=1, the network forms a tree structure without cycles. Because new nodes only ever attach to a single existing node, there are no competing topological forces pulling nodes into a dense, dynamic cluster. This often results in isolated branched structures that rapidly fragment upon attack.

ER Model (Exponential)

The Erdős–Rényi model connects pairs of nodes randomly. Any two nodes are connected with a fixed probability p.

This generates a homogenous degree distribution that decays exponentially for k ≫ ⟨k⟩.

Targeted Attack

Removes the highly-connected hubs. The target node v satisfies:

kv = maxi(ki)

Scale-free networks are robust against random failures but highly fragile to targeted attacks on hubs (m doesn't protect against this if hubs fall).
