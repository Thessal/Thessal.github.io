# Scale-Free Network Stability & Phase Transitions

This repository contains an interactive simulation of **Structural Phase Transitions** within complex networks.

## 1. The Research Context

While these networks are resilient to random shocks, they are uniquely vulnerable to **targeted attacks**. This simulation visualizes the **structural vulnerability** (specifically, **hub-centric fragility**) of preferential attachment networks: the moment when removing a critical hub triggers a global collapse of the **Giant Component ($P_\\infty$)**.

## 2. Interactive Simulation

The included `index.html` (hosted via GitHub Pages) provides a real-time dashboard to explore these dynamics:

* **Engine:** Barabási–Albert (BA) model with tunable preferential attachment.
* **Attack Vector:** Sequential removal of nodes based on **Degree Centrality**.
* **Metrics:** Real-time tracking of the phase transition threshold  and log-log degree distribution plots.

## 3. The Workflow

1. **Mathematical Derivation:** Translating network theory into simulation logic.
2. **Code Synthesis:** Generating D3.js visualizations.
3. **Validation:** Automated error-checking for look-ahead bias and topological consistency.
